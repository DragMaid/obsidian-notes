# Analog Signals
+ Emit data in a continuous function form 
+ Easily distorted by noises
+ Use wide range of values 
+ Lower bandwidth than [[2024-10-07_Digital-Signals|Digital Signals]]
+ Example: radio, sound, temperature, speed
