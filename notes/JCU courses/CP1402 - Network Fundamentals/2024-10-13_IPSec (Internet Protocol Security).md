# IPSec (Internet Protocol Security)
+ An Encryption protocol - defines rues for encryption, authentication, key management for TCP/IP transmissions
+ Creates secure connections in 5 steps:
    1. IPSec initiation
    2. Key management
    3. Security negotiations
    4. Data transfer
    5. Termination
+ Operates in 2 modes:
    1. Transport mode
    2. Tunnel mode
