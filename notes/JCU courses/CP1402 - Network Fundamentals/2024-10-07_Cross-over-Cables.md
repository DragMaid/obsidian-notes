# Cross over Cables
+ Sometimes called patch cable
+ Use [[Cooper and fiber wires|twisted-pair cable]]
+ Both end connectors should be the different type (eg. T-568A - T-568B)
+ Used to connect devices of the same type(switch - router, ...)
